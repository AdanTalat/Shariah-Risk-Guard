# Governance_Project_Islamic_Banking
This is the project for our 6th semester course of cybersecurity and governance for banking sector, particularly for islamic banks.

# Background
We have developed a Governance, Risk, and Compliance (GRC) prototype called ShariahGuard.

It maps SBP cybersecurity regulations, PECA, ISO 27001, NIST CSF, and Islamic banking governance requirements into measurable Key Risk Indicators. Instead of manually reviewing compliance, the tool calculates an overall regulatory risk score and generates dashboards and PDF reports to help management understand their security posture.


## Steps to run the app:
Create a Virtual Environment
```
python -m venv venv
```
Activate the Virtual Environment
```
venv\Scripts\activate
```
Install Requirements
```
pip install -r requirements.txt
```
Run the Application
```
streamlit run app.py
```
